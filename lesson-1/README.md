# Lesson 1: Python Thinks Like a Hacker

This folder contains all the teaching materials for the first lesson of the Python + Cybersecurity course.

## 📂 Contents

- **notebooks/**: Interactive Jupyter Notebooks for students.
  - `Lesson_1_EN.ipynb`: English version.
  - `Lesson_1_NO.ipynb`: Norwegian version.
- **slides/**: Presentation slides for the instructor.
  - `Lesson_1_EN_Slides.md`: English slides.
  - `Lesson_1_NO_Slides.md`: Norwegian slides.
- `requirements.txt`: Python dependencies needed for the lesson.
- `lesson-plan.md`: The original lesson plan.

## 🚀 Getting Started

### Prerequisites

1.  **Install Python:** Download and install Python from [python.org](https://www.python.org/).
2.  **Install VS Code:** Download from [code.visualstudio.com](https://code.visualstudio.com/).
3.  **Extensions:** Install the following extensions in VS Code:
    -   **Python** (Microsoft)
    -   **Jupyter** (Microsoft)
    -   **Marp for VS Code** (Marp Team) - *Optional, for viewing slides as a presentation.*

### Installation

Open a terminal in this folder and run:

```bash
pip install -r requirements.txt
```

### Running the Lesson

1.  Open `notebooks/Lesson_1_EN.ipynb` (or NO) in VS Code.
2.  Select the Python kernel (top right corner).
3.  Run code cells using `Shift + Enter`.

### Viewing Slides

1.  Open `slides/Lesson_1_EN_Slides.md`.
2.  Click the "Open Preview to the Side" icon (top right).
3.  To present, click the Marp icon in the toolbar and select "Toggle Marp Preview" or export to PDF/HTML.
